## Structure

Lyra will support genres in phases:

- **Phase 1 (MVP)**: 10 main genres
- **Phase 2**: 30-50 Electronic subgenres
- **Phase 3**: 100-200 detailed subgenres
- **Phase 4+**: Emerging and micro-genres

## Maintenance Guidelines

### Adding New Genres

When adding a new genre to this list:

1. **Verify existence**: Check if it's used on Spotify, Beatport, or other music platforms
2. **Define characteristics**: What makes it unique?
3. **Find examples**: List 2-3 representative artists/tracks
4. **Categorize**: Which parent category does it belong to?
5. **Update count**: Increment the total count at the top

### Removing Genres

Genres should rarely be removed, but if a genre is:
- Merged into another genre
- No longer in use
- Incorrectly categorized

Mark it as `[DEPRECATED]` rather than removing it entirely.

### Crossover Genres

Some genres appear in multiple categories (e.g., Bass House, Microhouse). In these cases:
- List in the primary category
- Note `(crossover)` in secondary categories
- Link to the primary listing
